# Panel de Control de Clientes en Tiempo Real 🚀
### Trabajo Práctico Integrador Grupal - Programación Visual 2026

Este proyecto consiste en el desarrollo de una aplicación web funcional de nivel profesional utilizando *React* y *Vite. La aplicación actúa como un Panel de Control que se conecta de forma asrincrónica a la API pública **FakeStore API* para la gestión de usuarios, abandonando el uso de datos en memoria local.

##Tecnologías y Librerías Utilizadas

* *Entorno de Desarrollo:* [React](https://react.dev/) inicializado mediante [Vite](https://vitejs.dev/) (con JSX).
* *Enrutamiento:* [React Router DOM](https://reactrouter.com/) para la navegación Single Page Application (SPA).
* *Diseño e Interfaz:* Framework de componentes UI (MUI / React Bootstrap) con maquetado responsivo.
* *Estado Global:* Context API combinado con localStorage para el control de accesos y persistencia de sesión.
* *Asincronismo:* Peticiones HTTP estructurales utilizando la sintaxis de async/await mediante fetch / Axios.
